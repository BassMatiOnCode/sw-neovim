Neovim

There are several operating modes:

  1. Normal
  2. Insert
  3. Visual
  4. Command-line

The normal mode is active when you open a file and navigate with the hjkl or the usual keyboard navigation keys or key combinations. This mode is used to navigate, delete, copy and paste. To return to normal mode, type ESC.

The insert mode is entered when you type i in normal mode. There are more (iIaAoO), look them up in the commad overview. Insert mode is used for inserting text. You can still delete text or navigate with the keyboard navigation keys (LEFT, RIGHT, UP, DOWN, PGUP, PGDN, HOME, END), but in general it is better to return to normal mode for that.

The visual mode is entered when you type v in the normal mode and is used to select text with the navigation keys to cut or copy or perform other actions.

The command line mode is entered when you type : in the normal mode. It is used to enter commands and execute them, such as write (w), quit (q) and search and replace.

The current mode is indicated in the bottom line, it shows --insert-- when in insert mode, --visual-- when in visual mode, a block cursor when in command mode or just nothing when in normal mode.

Each mode comes with a specific set of commands.

The screen is divided vertically into sections and the command line at the bottom. Each section displays a file. The command line is used to enter Neovim commands. Pressing ESC on the keyboard takes you to the command line. Then you can type a sequence of characters to enter the command. When finished, press ENTER on the keyboard to execute the command.

The command :q! (quit forced) will take you to the exit without saving changes. The command :wq means "write and quit".

The cursor in normal (command) mode is a block cursor. In insert mode, it is a vertical bar.

Commands are case-sensitive!

  1. Start Neovim

    There is a single line at the bottom of the window, called command line. Typing : takes you to the command line. Then you can enter commands, a sequence of characters, terminated by pressing the Enter key.

  2. Type :help

    This opens the (read-only) help document at the top of the window. You can navigate it the document with the usual arrow keys on the keyboard. The combination Ctrl+Pos1 takes you to the top of the document. You can doubleclick on links that carry you to another location in the document. You can doubleclick on any word to jump to its explanation.

  3. Type :q to quit working on the help document. You are taken "back" to the empty file that was automatically created upon the Neovim start.

Commands in Normal Mode

Some commands can be combined with a move command: CMD MOVE. For example,

d
(delete), and MOVE is one of e (end of word), w (start of next word) or $ (end of line). If you type the move argument multiple times, the cursor will move accordingly. For example, try deee.

Commands in Visual Mode

To enter character selection mode, type v in normal mode. Use the navigation keys or key combinations to select text. Then press a command key to execute a command on the selection. You will automatically be returned to normal mode then.

To enter line selection mode, type SHIFT+v in normal mode. Use the navigation keys to select entire lines of text.

To enter visual block mode, type CTRL+v in normal mode. Then use navigation keys to select a block of text. This is useful when working with tables.

Search and Replace

The following commands are given in standard mode:

Commands in normal mode
/ Search forward
? Search backwards
// Search again
* Search forward word under cursor
# Search backward word under cursor
g* Search forward partial word under cursor
g# Search backward partial word under cursor

Use CTRL+o to return to the position before the search.
